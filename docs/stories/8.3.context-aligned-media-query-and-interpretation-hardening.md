## Story

**As a** platform owner,  
**I want** a clearer and more assertive media query and interpretation pipeline,  
**so that** the system mitigates contextual misunderstanding when searching for elements to build the video.

---

## Acceptance Criteria

1. The system defines an explicit transformation chain from narrative scene -> visual brief -> query strategy -> candidates -> score -> operational decision.
2. Query generation uses structured scene context, visual function, restrictions and literalness instead of shallow keyword extraction.
3. Low-confidence results (`<60`) generate diagnostic feedback that informs the next query attempt.
4. The system persists query attempts and the reasons for reformulation.
5. The review queue only receives candidates that have crossed the minimum plausibility threshold for human judgment.
6. The implementation is documented well enough to support future tuning of prompt/query strategies.
