## Story

**As an** operator,  
**I want** the system to generate thumbnails and reusable cover assets for video distribution,  
**so that** videos can be published with visually strong, branded and platform-ready thumbnail imagery.

---

## Acceptance Criteria

1. The system can generate a thumbnail asset per production using template-aware branding and variation rules.
2. The thumbnail pipeline supports at least title/headline, branding, background/hero media and optional avatar/presenter treatment.
3. The system stores generated thumbnail metadata and resulting file references.
4. Generated thumbnail assets can be reused by downstream publishing flows, including YouTube custom thumbnails and social cover/media metadata where applicable.
5. The thumbnail generation flow is documented with its input contract, output contract and extensibility points.
