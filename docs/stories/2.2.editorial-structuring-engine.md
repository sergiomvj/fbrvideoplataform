## Story

**As a** production system,  
**I want** to transform raw editorial input into a template-compatible narrative structure,  
**so that** downstream visual planning and composition operate on deliberate blocks instead of raw text.

---

## Acceptance Criteria

1. The engine accepts normalized production input and outputs a structured narrative base with ordered scenes or blocks.
2. The structuring result includes role/purpose of each scene, text or narration content and estimated duration.
3. The result respects the selected template's constraints and intent.
4. Automatic mode and manual mode both result in the same narrative-base contract, even if their inputs differ.
5. Invalid or non-structurable inputs fail with explicit domain-level feedback rather than generic transport errors.
6. The output contract is stable and documented for visual planning and composition stories.
