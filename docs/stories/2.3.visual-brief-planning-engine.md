## Story

**As a** media intelligence pipeline,  
**I want** every scene to be translated into a structured visual brief,  
**so that** sourcing, scoring and human review work from explicit intent rather than interpretation of raw narration.

---

## Acceptance Criteria

1. Every narrative scene/block can generate a visual brief with the minimum fields defined in the SPEC.
2. Visual briefs explicitly include function, subject, contextual constraints, allowed/prohibited elements and preferred asset type.
3. Briefs are valid for both automatic sourcing and manual review/override use cases.
4. Brief generation preserves clear traceability from scene -> brief.
5. Invalid or underspecified scenes fail with actionable feedback.
