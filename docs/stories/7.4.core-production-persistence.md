## Story

**As a** platform owner,  
**I want** productions and workflow transitions persisted in PostgreSQL,  
**so that** the system survives restarts and preserves complete operational history.

---

## Acceptance Criteria

1. Production root data is persisted in PostgreSQL.
2. Normalized input snapshots are persisted per production.
3. Workflow state transitions are persisted append-only.
4. Listing and retrieval endpoints no longer depend on `_productions_store`.
5. The aggregate rules remain intact after persistence is introduced.
6. Migration and repository strategy are documented sufficiently for downstream work.
