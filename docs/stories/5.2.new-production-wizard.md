## Story

**As a** production operator,  
**I want** a guided wizard for starting new productions,  
**so that** I can choose mode, template, variation and input confidently without needing to understand backend details.

---

## Acceptance Criteria

1. The wizard supports explicit selection of mode, template and variation.
2. The wizard supports the defined input types for production creation.
3. Manual mode exposes the additional asset-related input areas required by the product.
4. The wizard consumes backend template metadata rather than duplicating template rules in the UI.
5. The resulting submission creates a production through the secured gateway and shows clear success/failure states.
