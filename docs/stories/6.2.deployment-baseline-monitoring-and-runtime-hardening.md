## Story

**As a** platform owner,  
**I want** the deployment baseline and runtime hardening in place early,  
**so that** all later implementation runs on an observable and secure operational substrate.

---

## Acceptance Criteria

1. The baseline deployment topology is documented and reproducible for local/staging environments.
2. Monitoring hooks for Prometheus/Grafana are wired at the baseline level.
3. Secret and environment handling follows the project security rules.
4. Reverse proxy/runtime exposure is hardened for the planned service topology.
5. The runtime baseline is ready to host later backend, frontend, n8n and OpenClaw integrations without redesign.
