## Story

**As a** product owner,  
**I want** the frontend baseline cleaned and aligned with quality gates and standards,  
**so that** later UI remediation stories build on a trustworthy and conformant base.

---

## Acceptance Criteria

1. `npm run lint` passes.
2. Navigation does not expose routes that do not exist.
3. Base layout metadata and typography align with approved standards or are explicitly documented as a temporary exception.
4. Known structural frontend lint issues are fixed.
5. The baseline shell remains functional after cleanup.

---

## Tasks / Subtasks

- [ ] Task 1: Fix current lint failures (AC: 1, 4)
  - [ ] Remove empty-interface violations
  - [ ] Fix effect/setState misuse
  - [ ] Remove unused imports and similar structural issues

- [ ] Task 2: Repair navigation baseline (AC: 2, 5)
  - [ ] Remove or gate dead links
  - [ ] Keep shell navigation aligned with actual routes

- [ ] Task 3: Align layout baseline (AC: 3)
  - [ ] Update fonts and metadata
  - [ ] Document any temporary standard deviations if needed
