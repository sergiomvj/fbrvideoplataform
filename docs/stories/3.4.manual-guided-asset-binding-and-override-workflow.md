## Story

**As a** production operator,  
**I want** to bind and override assets manually in a guided way,  
**so that** the system can support sensitive or controlled productions without breaking structural discipline.

---

## Acceptance Criteria

1. Manual mode supports explicit asset binding to scenes or blocks.
2. Operators can mark assets as required, fixed or prohibited where the mode allows it.
3. Manual bindings remain subject to template compatibility rules.
4. Manual override actions are auditable and persist in the production record.
5. Downstream composition uses the manual bindings as authoritative when present.
