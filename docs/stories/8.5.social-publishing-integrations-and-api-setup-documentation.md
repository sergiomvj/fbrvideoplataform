## Story

**As a** platform owner,  
**I want** integrations for YouTube, Instagram and Facebook plus setup documentation for their APIs,  
**so that** finished videos and thumbnails can be distributed through supported social channels in a controlled and repeatable way.

---

## Acceptance Criteria

1. The system defines canonical publishing contracts for YouTube, Instagram and Facebook.
2. Each platform integration is isolated behind an adapter or provider-specific publishing service.
3. The publishing flow supports asset handoff, metadata mapping, remote publish attempt tracking and terminal result handling.
4. YouTube publishing supports video upload and custom thumbnail handoff where credentials and permissions allow it.
5. Instagram and Facebook publishing flows are defined in accordance with professional/business account constraints and supported API paths.
6. A documentation guide exists with step-by-step API setup instructions for YouTube, Instagram and Facebook, including required app/account prerequisites, permissions and operational notes.
