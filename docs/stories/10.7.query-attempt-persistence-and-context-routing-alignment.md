## Story

**As a** media intelligence owner,  
**I want** Story 8.3 completed through persisted query attempts and real routing integration,  
**so that** context-aligned sourcing decisions are auditable and actually govern which candidates reach review.

---

## Acceptance Criteria

1. Query attempts, diagnostics and reformulation reasons are persisted and linked to production/scene/brief.
2. The media sourcing path uses the explicit query-building flow rather than bypassing it.
3. Low-confidence results trigger reformulation behavior before human review eligibility.
4. Review eligibility is enforced by the routing/decision layer according to the documented threshold logic.
5. Story 8.3 can be re-reviewed as compliant against its original acceptance criteria.
