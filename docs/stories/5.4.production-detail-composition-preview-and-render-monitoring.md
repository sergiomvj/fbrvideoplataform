## Story

**As a** production operator,  
**I want** a production detail view with composition preview and render monitoring,  
**so that** I can understand the full state of a production and trust what is being sent to render.

---

## Acceptance Criteria

1. The production detail view shows lifecycle status, mode, template and variation.
2. The screen exposes composition preview data in a readable operational format.
3. Render status and result visibility are presented clearly.
4. The screen reflects failures or blocked states without requiring backend log inspection.
5. The detail view is consistent with the operational shell and navigation patterns.
