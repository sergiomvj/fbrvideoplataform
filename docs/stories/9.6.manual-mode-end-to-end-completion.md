## Story

**As a** production operator,  
**I want** manual mode to work end-to-end with pre-flow asset input, authoritative bindings and template-safe composition,  
**so that** controlled productions can be created without breaking system structure.

---

## Acceptance Criteria

1. Manual mode in the wizard exposes the additional asset-related inputs required by product documents.
2. The backend exposes create/list/remove manual binding APIs.
3. Manual bindings are persisted and auditable.
4. Manual bindings enforce template compatibility rules.
5. Downstream composition uses manual bindings as authoritative when present.
6. Production creation and later composition paths clearly distinguish automatic from manual mode.
